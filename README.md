MyMichlin is an iOS app developed as part of the FIT3175 Mobile Application Development unit.
It is a restaurant discovery application inspired by Michelin-style listings, showcasing my ability to design and build a complete iOS application using Swift, Storyboard, MVC architecture, and multiple iOS frameworks.

This project achieved a High Distinction (95%), meeting all requirements for UI design, functionality, code quality, and technical implementation.

**Note:** This app relies on external APIs (Gemini, Google Places, Google Maps). These API keys have expired and have been removed for security. The project will not run without valid keys. The repo is for reference purposes. For details or a walkthrough, feel free to contact me.
**Disclaimer** Google Place API does not permit local caching of restaurant information. This app temporarily caches restaurants' information to showcase knowledge for academic purposes.

## Technical Features

### 1. Restaurant Search & Discovery
- Implemented Michelin-style restaurant search using **Google Places API**
- Created user-friendly restaurant cards with detailed information display

### 2. Map View & Location Services
- Developed interactive maps using **MapKit** and **Google Maps SDK**
- Implemented custom restaurant markers and detailed map annotations

### 3. Restaurant Details & Ratings System
- Designed comprehensive detail views showing reviews, photos, and cuisine information
- Created smooth transitions and intuitive UI interactions

### 4. AI-Powered Recommendations
- Integrated **Google Gemini API** for intelligent restaurant suggestions
- Built a personalised recommendation engine based on user preferences

### 5. UI/UX Implementation
- Developed **Storyboard-based** interface with programmatic constraints
- Followed **Apple Human Interface Guidelines** for consistency
- Implemented custom animations, consistent spacing, and responsive layouts
